# bioedu_ggoncoplot

Tutorial on making your first oncoplot.


1. Download the [workshop materials](https://github.com/selkamand/bioedu_ggoncoplot/archive/refs/tags/Mar31.zip)
2. Unzip
3. Open the `bioedu_oncoplot.Rproj` file using Rstudio
4. Open `R/first_oncoplot.qmd` and work through top to bottom
